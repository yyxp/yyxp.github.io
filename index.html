Web 中文字体处理总结

背景介绍

Web 项目中，使用一个合适的字体能给用户带来良好的体验。但是字体文件太多，如果想要查看字体效果，只能一个个打开，非常影响工作效率。因此，需要实现一个功能，能够根据固定文字以及用户输入预览字体。在实现这一功能的过程中主要解决两个问题：

  • 中文字体体积太大导致加载时间过长
  • 字体加载完成前不展示预览内容

现在将问题的解决以及我的思考总结成文。

观视频2019「答案」年终秀有感

发表于 2020-01-10 | 分类于 随记 
2019年香港的一系列事件吸引了我的注意力，为了吃瓜，渐渐的，我了解到了观视频工作室，
里面各位大佬的视频、演讲开阔了我的眼界，进而让我对社会科学，政治经济学产生了极大的兴趣，
以后我一定要学习一下社会科学，用大数据的方法实践研究，做一个合格的云学生。

观看观视频2019「答案」年终秀系列视频后给我的意识：

1.中国发展很迅速，存在的社会问题也很多，需要辩证的看，在发展中解决发展的问题

2.社会科学很重要，一定要学，香港问题一定程度上吃了社会科学研究落伍的亏

3.毛主席的预言真准，以后一定要多多阅读下毛主席的语录和诗词

4.当前的世界正处于百年未有之大变局，中国的机会很多，年轻人的机会很多，要拼搏，干就完事儿了

5.姓资还是姓社？当然还是社会主义大法好，尤其是中国特色社会主义。知道为何坚持经济建设为中心，一百年不动摇吗

6.乳法，法国是个大国。原来法国对欧洲历史及人类有着这么大的影响，英国真是8行

7.金融开放的挑战和机遇，未来属于谁

8.怎样理解政治制度？范勇鹏老师的课，该上一上，我们需要了解这个社会的各种政治体制

9.多了解了解各个国家吧，多看看历史，有了国际视野，生活和工作中的事情就渺小易操作了

以上只是随记，以后想起啥就补上。

毕业多年，不曾为国家和社会做出啥贡献，想着努力工作好，过好生活，便是最大的贡献。愿祖国繁荣昌盛！
